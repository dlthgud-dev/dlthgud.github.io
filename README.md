# dlthgud.github.io

## Jekyll

### 루비 설치하기

```
sudo apt-get install ruby-full
```

### Jekyll 설치하기

```
gem install bundler jekyll
```

### Jekyll 준비하기

```
jekyll new my-awesome-site
```

### Jekyll 실행하기

```
cd my-awesome-site
# bundle exec jekyll serve
jekyll server --host 0.0.0.0
```
